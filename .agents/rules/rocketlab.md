Você foi designado(a) para desenvolver um módulo de uma aplicação (Frontend e Backend) de
Sistema de Gerenciamento de E-Commerce. O usuário é o Gerente da loja e o sistema permitirá
que o usuário veja a lista dos seus produtos, e acesse o desempenho de vendas e avaliações
de cada produto, e consiga modificar os produtos.

O projeto deve ser criado com a seguinte stack:

● Frontend: Vite - React + Typescript
● Backend: FastAPI (Python)
● Banco de dados: SQLite

Materiais: Para esta tarefa, serão fornecidos alguns recursos para que a atividade tenha foco
nos conceitos passados em aula. Você receberá inicialmente:

1. Um repositório contendo a estrutura base do projeto, o banco de dados modelado no
SQLAlchemy (ORM) e a ferramenta de migração de bancos de dados Alembic;

2. Os arquivos .csv contendo os dados para população das tabelas.

Você deverá desenvolver e versionar o projeto no GitHub com um README contendo o passo a
passo para executar a aplicação.

Requisitos:

● Eu gostaria de poder ter produtos com informações associadas;
● Eu gostaria de poder navegar em um catálogo dos produtos;
● Eu gostaria de poder acessar mais detalhes de cada produto, como medidas, vendas e avaliações feitas pelos consumidores;
● Eu gostaria de poder buscar um ou mais produtos específicos;
● Eu gostaria de poder adicionar, remover e atualizar os produtos individualmente;
● Eu gostaria de poder ver a média das avaliações de um produto.

Você pode explorar a sua criatividade com outras funcionalidades como tratamento de erros,
documentação, testes automatizados, autenticação, paginação via backend, filtros de busca,
responsividade, caching de consultas, resposta a avaliações...
