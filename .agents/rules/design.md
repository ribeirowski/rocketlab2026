Usar tailwindCSS.

Usar componentes atômicos.

Tipografia: Use HelveticaNeue com escala modular
- Headers: Bold e escuros (Dark Teal).
- Body: Regular para legibilidade.
- Labels/Captions: Tamanhos menores para informações secundárias.

Hierarquia de Cores:
- Dark Teal #064C4F
- Slime Green #BBEA70
- Blaze Orange #FF7006
- Pansy purple #6C1144

Estados de Feedback:
Skeleton Screens: Em vez de um "Loading" genérico, use Skeletons para mostrar a estrutura da página enquanto os produtos carregam.
Feedback Instantâneo: Ao deletar ou editar um produto, utilize Toasts (notificações flutuantes).

Formulários de Edição:
Use Modais (Lighboxes) para edição rápida de produtos sem perder o contexto da lista.
Agrupe campos relacionados.

Espaçamento: Use múltiplos de 8px (8, 16, 24, 32) para margens e paddings.

Arredondamento (Border Radius): Use cantos arredondados para passar uma sensação moderna, mas mantendo o aspecto de ferramenta de trabalho.

Acessibilidade: Sempre verifique o contraste entre o texto e o fundo.
